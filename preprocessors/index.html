Що таке препроцесори?

CSS-препроцесори — це розширення CSS, які дозволяють використовувати змінні, вкладені правила, міксіни, функції та інші корисні можливості.

Вони допомагають зробити CSS більш структурованим, гнучким та повторно використовуваним.

Популярні препроцесори

  • LESS
    • Менш популярний зараз, але використовується в деяких проєктах.
    • Простий у вивченні, синтаксично дуже схожий на стандартний CSS.
  • SASS
    • Один із найпопулярніших препроцесорів.
    • Два синтаксиси: SASS (відступи замість фігурних дужок) і SCSS (майже як CSS, але з додатковими можливостями).
    • Має розвинуту екосистему та потужний набір можливостей.

Недоліки препроцесорів

  • Потрібна компіляція перед використанням у браузері.
  • Ускладнюється налагодження стилів (іноді важко знайти джерело проблеми).
  • Деякі можливості вже реалізовані у звичайному CSS (змінні, вкладеність, @import замінюється модулями CSS).
  • Вимагають вивчення додаткового синтаксису.

SASS

Браузер не розуміє код Sass. Тому вам знадобиться препроцесор Sass, щоб перетворити код Sass у стандартний CSS.

Цей процес називається транспіляцією. Отже, вам потрібно надати транспілятору (певній програмі) код Sass, а потім отримати назад код CSS.

Способи компіляції SASS

  • Live Sass Compiler (плагін для VSCode) — простий у використанні, підходить для початківців.
  • Node.js + npm пакет sass — більш професійний підхід для автоматизації.
  • Ruby SASS (застарілий) — історична згадка, зараз не використовується.

Ми будемо використовувати розширення для VSCode Live Sass Compiler.

Створення першого SASS файлу

Для роботи з SASS необхідно підготувати папку проєкту.

В папці проєкту необхідно створити папку scss

В цій папці необхідно створити файл style.scss.

Файли з кодом SASS майють розширення .scss

Після додайте до вашого файлу .scss наступний базовий код:

$color: #007acc;

body {
  background-color: $color;
}

Далі натисніть кнопку Watch Sass. У папці проєкту повинні з'явитися скомпільовані стилі у вигляді звичайного css-файлу.

Змінні SASS

Змінні — це спосіб зберігати інформацію, яку можна використовувати пізніше.

За допомогою Sass ви можете зберігати інформацію у змінних, наприклад:

  • рядки
  • чисел
  • кольори
  • логічні значення
  • списки
  • нулі

Sass використовує символ $, за яким слідує ім’я, для оголошення змінних:

$variablename: value;

Наступний приклад демонструє використання змінних SASS:

Приклад 
$myFont: Helvetica, sans-serif;
$myColor: red;
$myFontSize: 18px;
$myWidth: 680px;

body {
  font-family: $myFont;
  font-size: $myFontSize;
  color: $myColor;
}

#container {
  width: $myWidth;
}

Вкладеність SASS

Вкладені правила

Sass дозволяє вкладати селектори CSS так само, як і HTML.

Подивіться на приклад коду Sass для навігації сайтом:

nav {
    ul {
        margin: 0;
        padding: 0;
        list-style: none;
    }
    li {
        display: inline-block;
    }
    a {
        display: block;
        padding: 6px 12px;
        text-decoration: none;
    }
}

Цей SASS-код перекладеться у наступний CSS-код:

nav ul {
    margin: 0;
    padding: 0;
    list-style: none;
}
nav li {
    display: inline-block;
}
nav a {
    display: block;
    padding: 6px 12px;
    text-decoration: none;
}
Приклад

Оскільки ви можете вкладати властивості в Sass, він чистіший і легший для читання, ніж стандартний CSS.

Вкладені властивості

Багато властивостей CSS мають однаковий префікс, наприклад font-family, font-size і font-weight або text-align, text-transform і text-overflow.

За допомогою Sass ви можете записати їх як вкладені властивості:

body {
    font: {
        family: $myFont;
        size: $myFontSize;
    }

    color: $myColor;
}

Цей код буде перекладений в CSS наступним чином:

body {
    font-family: Helvetica, sans-serif;
    font-size: 18px;
    color: red;
}

@import

Sass зберігає CSS-код DRY (Don't Repeat Yourself - не повторюйте себе). Один із способів написання СУХОГО коду - зберігати відповідний код в окремих файлах.

Ви можете створювати невеликі файли з фрагментами CSS для включення в інші файли Sass. Прикладами таких файлів можуть бути: файл скидання, змінні, кольори, шрифти, розміри шрифтів тощо.

Директива @import дозволяє включати вміст одного файлу в інший.

Директива CSS @import має серйозний недолік через проблеми з продуктивністю; він створює додатковий запит HTTP кожного разу, коли ви його викликаєте. Однак директива Sass @import включає файл у CSS; тому під час виконання не потрібен додатковий виклик HTTP!

Синтаксис:

@import "filename";

Порада: вам не потрібно вказувати розширення файлу, Sass автоматично припускає, що ви маєте на увазі файл .sass або .scss. Ви також можете імпортувати файли CSS. Директива @import імпортує файл, і будь-які змінні або міксини, визначені в імпортованому файлі, потім можна використовувати в основному файлі.

Створимо файл _reset.scss, який буде прибирати стилі за замовчуванням:

html,
body {
  margin: 0;
  padding: 0;
}

Також створимо файл _variables.scss в який перенесемо створенні змінні:

$myFont: Helvetica, sans-serif;
$myColor: red;
$myFontSize: 18px;
$myWidth: 680px;

Імпортуємо створені файли у основний файл стилів:

@import "_reset";
@import "_variables";

body {
    font: {
        family: $myFont;
        size: $myFontSize;
    }

    color: $myColor;
}

#container {
    width: $myWidth;
}

Результат у файлі CSS:

html,
body {
  margin: 0;
  padding: 0;
}

body {
  font-family: Helvetica, sans-serif;
  font-size: 18px;
  color: red;
}

#container {
  width: 680px;
}

Зверніть увагу, що при використанні символу _ на початку імені файлу повідомляє компілятору, що цей файл не треба компілювати в окремий CSS-файл. Весь вміст таких файлів компілюється прямо в основний файл стилів.

Міксіни

Директива @mixin дозволяє створювати код CSS, який буде повторно використовуватися на веб-сайті.

Директива @include створена, щоб дозволити вам використовувати (включати) міксин.

Синтаксис:

@mixin name {
    property: value;
    property: value;
    ...
}

У наступному прикладі створюється міксин під назвою «important-text»: 

@mixin important-text {
    color: red;
    font-size: 25px;
    font-weight: bold;
    border: 1px solid blue;
}

Щоб використати створений міксін необхідно використати директиву @include:

.danger {
    @include important-text;
    background-color: green;
}

У звичайний CSS цей приклад перекладеться наступним чином:

.danger {
    color: red;
    font-size: 25px;
    font-weight: bold;
    border: 1px solid blue;
    background-color: green;
}

Передача змінних у міксини

Міксіни можуть приймати аргументи.

/* Define mixin with two arguments */
@mixin bordered($color, $width) {
    border: $width solid $color;
}

.myArticle {
    @include bordered(blue, 1px);  // Call mixin with two values
}

.myNotes {
    @include bordered(red, 2px); // Call mixin with two values
}

Ви можете задати значення за замовчуванням для параметрів міксіну:

@mixin bordered($color: blue, $width: 1px) {
    border: $width solid $color;
}

В такому випадку вам необов'язково вказувати всі параметри при включенні міксіну:

.myTips {
    @include bordered($color: orange);
}

Функції SASS

Функції в SASS — це вбудовані методи, які дозволяють виконувати різноманітні операції над значеннями прямо у CSS-коді. Вони допомагають спрощувати обробку кольорів, текстів, чисел та інших даних, роблячи код гнучкішим і ефективнішим.

Для чого використовуються:

  • Автоматичні обчислення розмірів.
  • Маніпуляція кольорами (освітлення, затемнення).
  • Форматування рядків.
  • Конвертація одиниць виміру.
  • Робота з масивами (списками) та картами (асоціативними масивами).

Далі ми розберемо декілька корисних функцій. Повний перелік функцій можна знайти в документації або на сайті W3School

darken() / lighten()

Змінює яскравість кольору.

$main-color: #007acc;

.button {
  background: darken($main-color, 10%);
}

Результат CSS:

.button {
    background: #005a9c;
}

mix()

Змішує два кольори в заданій пропорції

$color1: #ff0000;
$color2: #0000ff;

.element {
    background: mix($color1, $color2, 50%);
}

Результат CSS:

.element {
    background: #800080; /* Фіолетовий */
}

nth()

Повертає значення певного елемента зі списку.

$list: red, green, blue;

.color {
    color: nth($list, 2);
}

Результат CSS:

.color {
    color: green;
}

percentage()

Конвертує число у відсоткове значення.

.width {
    width: percentage(0.5);
}

Результат CSS:

.width {
    width: 50%;
}

unit()

Повертає одиницю виміру для числа.

$value: 10px;

.unit {
    content: unit($value);
}

Результат CSS:

.unit {
    content: "px";
}
