Анімації

CSS-анімації підходять для виконання простих рухів елемента, а також коли не потрібна сумісність із застарілими браузерами.

Ефект рухомих елементів на сайті можна створити за допомогою властивостей CSS3:

  • переходів — transition
  • анімації — animation
  • трансформації — transform. Слід зазначити, що трансформація не створює анімацію, лише трансформує елемент (переміщає, обертає, збільшує). Трансформації часто використовуються разом із властивостями animation і transition.

Властивість transition

CSS-переходи (transition), дозволяють CSS-властивостям змінювати одне значення на інше, за певний проміжок часу. Найпростіший приклад подібної поведінки — це зміна кольору фону кнопки-посилання за допомогою псевдокласу :hover, при якому одразу відбувається зміна значень CSS-властивості background-color

Приклад

Синтаксис: 

transition: [властивість] [час] [функція] [затримка];
  • [властивість] - вказуємо назви властивостей, зміни яких повинні бути анімованими. Можна вказати значення all.
  • [час] - за який відрізок часу стан властивості повинен змінитись, тобто це час анімації.
  • [функція] - функція, яка визначає швидкість зміни властивості. Наприклад, ease-in означає, що зміна розпочнеться повільно, а під кінець прискориться. А функція linear означає, що швидкість зміни властивості буде однаковою весь час.
  • [затримка] - визначає затримку перед початком переходу.
Робимо разом!

Ви можете вказати декілька переходів через кому.

transition: background-color 0.8s ease 0s, font-size 1s linear 0.2s;

Властивість transform

Властивість transform дозволяє трансофрмувати (тобто змінити) елемент. Ви можете його змістити, повернути та змінити розміри.

Трансформації бувають двох видів - 2D та 3D.

Часто трансформації використовуються разом із переходами. Переходи дозволяють зробити процес трансформації елемента плавним.

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: rotate(45deg);
}

2D трансформації

Трансформація відбувається у двох осях: X (по горизонталі) та Y (по вертикалі).

translate

translate(x, y) — зміна розташування елемента на значення x зліва направо, на значення y — зверху вниз. Якщо вказуються від'ємні значення, зсув буде спрямований справа наліво і знизу вгору 

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: translate(25px, 35px);
}

scale

scale(x, y) — масштабує елемент. Якщо вказані значення від 0 до 1, то елемент зменшується. Якщо значення більші за 1, елемент збільшується в масштабі. При від'ємних значеннях елемент відображає дзеркально. Слід зазначити, що при збільшенні елемента, якість відображуваного елемента може погіршитися. Діє для елементів блочного типу. Чому тільки для таких елементів? Оскільки масштабування — це одночасне збільшення ширини та висоти елемента. Змінювати ширину/висоту елемента можна для елементів з блочним або рядково-блочним представленням. Ширина та висота рядкових елементів обчислюється відповідно до вмісту рядкового елемента.

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: scale(2, 0.5);
}

rotate

rotate(xdeg) - повертає елемнт на зазначену кількість градусів. Додатні значення обертають за годинниковою стрілкою, від'ємні - проти.

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: rotate(45deg);
}

skew

skew(x, y) - дозволяє "нахилити" елемент.

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: skew(25deg, 30deg);
}

3D трансформації

3D трансформації дозволяють обертати елемент навколо одної з вісей (x, y, z)

rotateX

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: rotateX(45deg);
}

rotateY

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: rotateY(45deg);
}

rotateZ

.div {
    width: 100px; 
    height: 100px;  
    background-color: black;
    transform: rotateZ(45deg);
}

Використання transform та transition

Для того щоб досягнути ефекту анімації transform та transition можна використовувати разом.

.example {
    width: 100px;
    height: 100px;
    transition: all 0.3s ease-in-out 0s;
    background-color: black;
}
.translate:hover {
    transform: translate(25px, 35px);
}
.scale:hover {
    transform: scale(2, 0.5);
}
.rotate:hover {
    transform: rotate(45deg);
}
.skew:hover {
    transform: skew(25deg, 30deg);
}

Властивість animation та правило keyframes

Найпростіший вид анімації — це два кадри: початковий та кінцевий. Браузер розумітиме, що необхідно між цими двома кадрами анімувати елемент, залежно від заданих умов.

Кадри створюються за допомогою спеціального правила @keyframes. Самі кадри оголошуються ключовими словами: from — «від» або початковий, перший кадр, і to — «до» або кінцевий кадр, другий та останній кадр. Якщо кількість кадрів більше, ніж два, то до кадру звертаються, як до відрізків, вираженим у відсотках пройденого шляху. Наприклад, три кадри: 0% — початковий (перший кадр), 50% — другий кадр, 100% — кінцевий (третій кадр).

@keyframes name{
    from{
       початкові властивості
    }
    to{
        кінцеві властивості
    }
}
