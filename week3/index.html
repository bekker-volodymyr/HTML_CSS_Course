Посилання: тег <a>

Посилання

Посилання дозволяють переходити зі сторінки на сторінку.

Посиланням може виступати не тільки текст.

Для створення посилання обгорніть його в тег <a>. В атрибут href вкажіть шлях до іншої сторінки:

<a href="https://uk.wikipedia.org/wiki/HTML">Посилання на wikipedia</a>

Результат:

Посилання на wikipedia

Атрибут target

<a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>
  • _self - стандарт, відкриває сторінку в поточному вікні/вкладці
  • _blank - відкриває сторінку в новому вікні або вкладці
  • _parent - відкриває сторінку в батьківському фреймі
  • _top - відкриває в повноекранному режимі

  • Відвідане посилання — підкреслене та фіолетового кольору.
  • Активне посилання — підкреслене та червоного кольору.

Зображення: тег <img>

Для додавання зображення використовується тег <img>.

Цей тег є пустим.

Вимагає два атрибути: src - шлях до зображення та alt - альтернативний текст.

<img src="https://www.oxfordwebstudio.com/user/pages/06.da-li-znate/sta-je-html/sta-je-html.jpg" alt="HTML LOGO" >

Атрибут src

Атрибут src (source) - містить посилання на зображення. Це може бути абсолютне посилання на ресурс в інтернеті, або відносне посилання на ресурс всередині проєкту.

Атрибут alt

Текст вказаний в цьому атрибуті буде відображатись, якщо з зображенням виникли проблеми (повільне завантаження, помилка в шляху тощо), а також використовується читачами екрану.

Екранний читач — це програмне забезпечення, яке зчитує HTML-код і дозволяє користувачеві "слухати" вміст. Екранні читачі корисні для людей із порушеннями зору або труднощами в навчанні.

Значення атрибуту alt повинно описувати зображення.

Розміри зображення

Розміри зображення можна вказувати за допомогою атрибутів width та height.

Але краще використовувати стилі

img {

Властивість border-style

Визначає, який тип межі (бордера) відображати.

Доступні значення:

  • dotted — точкова межа
  • dashed — пунктирна межа
  • solid — суцільна межа
  • double — подвійна межа
  • groove — 3D межа з ефектом "гравіювання" (залежить від кольору межі)
  • ridge — 3D межа з ефектом "виступу" (залежить від кольору межі)
  • inset — 3D межа з ефектом "втиснення" (залежить від кольору межі)
  • outset — 3D межа з ефектом "випуклості" (залежить від кольору межі)
  • none — без межі
  • hidden — прихована межа

Властивість border-width

Визначає ширину всіх чотирьох меж (бордерів).

Можливі значення:

  • Конкретний розмір (наприклад, px, pt, cm, em тощо)
  • thin — тонка межа
  • medium — середня межа
  • thick — товста межа

Приклад стилізації:

thin
medium
thick
5px
0.5em

Властивість border-width може мати декілька значень для різних сторін (від одного до 4)

p.one {
    border-style: solid;
    border-width: 5px 20px; /* 5px top and bottom, 20px on the sides */
}
  
p.two {
    border-style: solid;
    border-width: 20px 5px; /* 20px top and bottom, 5px on the sides */
}
  
p.three {
    border-style: solid;
    border-width: 25px 10px 4px 35px; /* 25px top, 10px right, 4px bottom and 35px left */
}

Властивість border-color

Вказує на колір рамки

p.one {
    border-style: solid;
    border-color: red green blue yellow; /* red top, green right, blue bottom and yellow left */
}
  
  p.two {
    border-style: solid;
    border-color: green;
}
  
p.three {
    border-style: dotted;
    border-color: blue;
}

Рамка: властивість border

CSS-властивість border визначає рамку навколо будь якого HTML-елементу. Це скорочення яке приймає три значення: ширину (border-width), стиль (border-style, обов'язковий), колір(border-color). 

p {
    border: 2px solid powderblue;
}

Закруглені кути: border-radius

Властивість border-radius дозволяє скруглити кути вашої рамки елементу

Відступи: padding та margin

Властивість padding визначає відстань між текстом та рамкою. 

p {
    border: 2px solid powderblue;
    padding: 30px;
}

Внутрішні відступи (Padding) для окремих сторін

CSS має властивості для задання внутрішніх відступів (padding) для кожної сторони елемента:

  • padding-top — відступ зверху
  • padding-right — відступ справа
  • padding-bottom — відступ знизу
  • padding-left — відступ зліва

Доступні значення:

  • Довжина (length) — задає відступ у px, pt, cm тощо.
  • Відсотки (%) — задає відступ у відсотках від ширини батьківського елемента.
  • inherit — успадковує відступ від батьківського елемента.

Примітка: Від’ємні значення не допускаються.

Приклад:

Це елемент із різними внутрішніми відступами. 
div {
  padding-top: 20px;
  padding-right: 40px;
  padding-bottom: 10px;
  padding-left: 30px;
}

Padding і ширина елемента

Властивість width у CSS визначає ширину області вмісту елемента. Ця область знаходиться всередині padding, border і margin (модель коробки).

Якщо у елемента встановлена конкретна ширина, то доданий padding збільшить загальну ширину елемента. Це часто є небажаним ефектом.

Приклад:

Тут елементу <div> задана ширина 300px, але його фактична ширина стане 350px (300px + 25px лівий padding + 25px правий padding).

div {
  width: 300px;
  padding: 25px;
}
Це div з шириною 300px та padding 25px.

Збереження фіксованої ширини за допомогою box-sizing

Щоб зберегти ширину елемента 300px незалежно від padding, можна використовувати властивість box-sizing: border-box;. Це дозволяє padding впливати лише на область вмісту, а не на загальну ширину елемента. 

div {
  width: 300px;
  padding: 25px;
  box-sizing: border-box;
}
Це div з шириною 300px та padding 25px, але без збільшення загальної ширини.

Властивість margin визначає простір за межами рамки.

p {
    border: 2px solid powderblue;
    margin: 50px;
}

Margin — індивідуальні сторони

CSS має властивості для встановлення відступів (margin) для кожної сторони елемента:

  • margin-top — верхній відступ
  • margin-right — правий відступ
  • margin-bottom — нижній відступ
  • margin-left — лівий відступ

Можливі значення:

  • auto — браузер сам розраховує відступ
  • length — задає відступ у px, pt, cm тощо
  • % — задає відступ у відсотках від ширини батьківського елемента
  • inherit — успадковує значення від батьківського елемента

Підказка: Дозволяється використовувати від’ємні значення.

Приклад стилізації:

p {
    margin-top: 100px;
    margin-bottom: 100px;
    margin-right: 150px;
    margin-left: 80px;
}
margin-left: 50px

Злиття відступів (Margin Collapse)

Верхні та нижні відступи елементів інколи зливаються в один, який дорівнює найбільшому з двох.

Це не стосується лівого та правого відступів! Тільки верхніх і нижніх!

Приклад:

Демонстрація злиття відступів:

h1 {
  margin: 0 0 50px 0;
}

h2 {
  margin: 20px 0 0 0;
}

У прикладі вище:

  • Елемент <h1> має нижній відступ 50px.
  • Елемент <h2> має верхній відступ 20px.

Здавалося б, відстань між <h1> і <h2> мала б бути 70px (50px + 20px), але через злиття відступів фактичний відступ становить лише 50px (найбільший з двох).

CSS background-image

Властивість background-image задає фонове зображення для елемента.

За замовчуванням зображення повторюється, щоб покрити весь елемент.

Приклад: Встановлення фонового зображення для сторінки

body {
  background-image: url("paper.gif");
}

Приклад: Погане поєднання тексту та фону

Цей приклад демонструє, як текст може стати важко читабельним через невдалий вибір фонового зображення.

body {
  background-image: url("bgdesert.jpg");
}

Це текст на фоні зображення. Важливо підбирати фон так, щоб текст залишався читабельним.

Приклад: Фонове зображення для окремих елементів

Фонове зображення можна встановлювати не тільки для всього документа, а й для окремих елементів, наприклад, для абзаців <p>:

p {
  background-image: url("paper.gif");
}

Це абзац із власним фоновим зображенням.

CSS background-repeat

За замовчуванням властивість background-image повторює зображення як по горизонталі, так і по вертикалі.
