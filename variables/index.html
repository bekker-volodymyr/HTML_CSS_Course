Для чого потрібні змінні та функція var()

Функція var() використовується щоб підставити значення змінної.

Змінні CSS мають доступ до DOM, що означає, що ви можете створювати змінні з локальною або глобальною областю видимості, змінювати змінні за допомогою JavaScript і змінювати змінні на основі медіа-запитів.

Змінні CSS можна використовувати, коли мова йде про кольори вашого сайту. Замість того, щоб копіювати та вставляти ті самі кольори знову і знову, ви можете розмістити їх у змінних.

Синтаксис

var (--name, value)
Значення Опис
name Обов'язкове. Назва змінної (повинна розпочинатись з двох --)
value Опціональне. Значення, яке буде використовуватись, якщо змінна не була знайдена.

Як var() працює?

Перш за все: змінні CSS можуть мати глобальну або локальну область видимості.

До глобальних змінних можна отримати доступ/використовувати їх у всьому документі, тоді як локальні змінні можна використовувати лише всередині селектора, де вони оголошені.

Щоб створити змінну з глобальною областю, оголосите її всередині селектора :root. Селектор :root відповідає кореневому елементу документа.

Щоб створити змінну з локальною областю видимості, оголосите її всередині селектора, який її використовуватиме. 

:root {
    --blue: #1e90ff;
    --white: #ffffff;
  }
  
  body { background-color: var(--blue); }
  
  h2 { border-bottom: 2px solid var(--blue); }
  
  .container {
    color: var(--blue);
    background-color: var(--white);
    padding: 15px;
  }
  
  button {
    background-color: var(--white);
    color: var(--blue);
    border: 1px solid var(--blue);
    padding: 5px;
  }

Переваги використання var() такі:

  • робить код легшим для читання (більш зрозумілим)
  • значно полегшує зміну значень кольорів

Щоб змінити синьо-білий колір на більш м’який синьо-білий, вам просто потрібно змінити значення двох змінних

Локальні змінні

Іноді ми хочемо, щоб змінні змінювалися лише в певному розділі сторінки.

Припустімо, що нам потрібен інший колір синього для елементів кнопки. Потім ми можемо повторно оголосити змінну --blue всередині селектора кнопки. Коли ми використовуємо var(--blue) всередині цього селектора, він використовуватиме локальне значення змінної --blue, оголошене тут.

Ми бачимо, що локальна змінна --blue замінить глобальну змінну --blue для елементів кнопки

button {
    --blue: #0000ff; /* local variable will override global */
    background-color: var(--white);
    color: var(--blue);
    border: 1px solid var(--blue);
    padding: 5px;
}

Також ми можемо визначити нову змінну локально в тому селекторі, де плануємо її використовувати. 

button {
    --button-blue: #0000ff; /* new local variable */
    background-color: var(--white);
    color: var(--button-blue);
    border: 1px solid var(--button-blue);
    padding: 5px;
}

Змінні у медіа-запитах

Тепер ми хочемо змінити значення змінної в медіа-запиті.

.container {
    --fontsize: 25px;
}

.container {
    color: var(--blue);
    background-color: var(--white);
    padding: 15px;
    font-size: var(--fontsize);
}
  
  @media screen and (min-width: 450px) {
    .container {
      --fontsize: 50px;
    }
}
