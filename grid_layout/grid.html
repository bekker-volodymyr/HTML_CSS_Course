Що таке Grid Layout

Grid Layout — це інструмент CSS для створення гнучких та структурованих макетів веб-сторінок за допомогою поділу на рядки та колонки.

На відміну від Flexbox, який краще підходить для вирівнювання елементів по одному напряму, Grid дозволяє працювати з двома вимірами одночасно.

Основні поняття

  • Грід-контейнер: батьківський елемент, на якому застосовується властивість display: grid;.
  • Грід-елементи: дочірні елементи грід-контейнера, які автоматично розташовуються в рядках і колонках.
  • Лінія сітки (Grid Line): лінія, що розділяє елементи сітки. Вона може бути горизонтальною чи вертикальною.
  • Трек сітки (Grid Track): простір між двома паралельними лініями сітки. Можна вважати, що це рядок або стовпець сітки.
  • Комірка сітки (Grid Cell): простір між двома сусідніми рядками і двома сусідніми стовпцями ліній сітки, аналогічно комірці в таблиці. Це ділянка, в якій можна розмістити контент. Комірка є найменшою одиницею сітки, на яку можна посилатися при позиціонуванні елементів сітки.
  • Ділянка сітки (Grid Area): будь-який простір між чотирма лініями сітки, може складатися зі скількох завгодно комірок. Ділянка сітки може бути такою ж малою, як одна комірка, або такою ж великою, як всі комірки сітки.

Грід-контейнер

Для початку необхідно задати контейнеру властивість display: grid;. Наприклад:

.grid-container-demo {
  display: grid;
  grid-template-columns: auto auto auto;
  background-color: dodgerblue;
  padding: 10px;
}

.grid-container-demo > div {
  background-color: #f1f1f1;
  border: 1px solid black;
  padding: 10px;
  font-size: 30px;
  text-align: center;
}

Цей код створює грід із трьома колонками рівної ширини.

Кожен дочірній елемент такого контейнеру розміщуватиметься в окремій комірці.

Властивості grid-template-columns та grid-template-rows

Властивість grid-template-columns визначає кількість стовпців у вашому макеті сітки і може визначати ширину кожного стовпця.

Значення є списком, розділеним пробілами, де кожне значення визначає ширину відповідного стовпчика.

Якщо ви хочете, щоб макет сітки містив 4 стовпчики, вкажіть ширину цих 4-х стовпчиків або auto, якщо всі стовпчики повинні мати однакову ширину. 

.grid-container {
  display: grid;
  grid-template-columns: auto auto auto auto;
}

Якщо ви маєте більше 4 елементів сітки в 4-х стовпчиках, сітка автоматично додасть новий рядок для розміщення елементів.

fr це одиниця вимірювання у CSS Grid, яка означає "частку від доступного простору" (fraction). Вона використовується для розподілу простору між рядками або стовпцями гріда.

fr розподіляє тільки вільний простір, тобто той, що лишився після віднімання фіксованих значень (наприклад, px, em тощо).

Властивість grid-template-rows визначає висоту кожного рядка. 

.grid-container {
  display: grid;
  grid-template-rows: 80px 200px;
}

Властивості gap, column-gap, row-gap

Визначають відступи між рядками та колонками.

  • column-gap - відступи між колонками.
  • row-gap - відступи між рядками.
  • gap - скорочення для задання відступів між колонками та рядками
.grid-gap-example {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 20px;
}

Цей код встановлює відступи у 20px між усіма грід-елементами.

Функція repeat

Функція repeat() дозволяє створювати фрагменти списку треків, які повто­рюють­ся. Це дозволяє записати в більш компактній формі велику кількість однакових за розмірами стовпців або рядків. 

.container {  
  grid-template-columns: repeat(3, 1fr);
}

Грід-елементи

Контейнер сітки містить один або кілька елементів сітки.

За замовчуванням контейнер містить по одному елементу сітки для кожного стовпця і кожного рядка, але ви можете змінити стиль елементів сітки так, щоб вони охоплювали кілька стовпців і/або рядків.

Властивості grid-column-start, grid-column-end

Властивість grid-column-start вказує, де починається елемент сітки.

Властивість grid-column-end вказує, де закінчується елемент сітки. 

.item1 {
  grid-column-start: 1;
  grid-column-end: 3;
}

Властивість grid-column

Властивість стовпчик сітки є скороченою назвою для властивостей початок стовпчика сітки та кінець стовпчика сітки.

Щоб розмістити елемент, ви можете посилатися на номери рядків або використовувати ключове слово «span», щоб вказати, скільки стовпців буде охоплено елементом. 

.item2 {
  grid-column: 2 / span 2;
}

Властивість grid-area

Властивість grid-area є скороченням для властивостей grid-row-start, grid-column-start, grid-row-end і grid-column-end.

Синтаксис: grid-row-start / grid-column-start / grid-row-end / grid-column-end.

Зробіть так, щоб «item4» починався з рядка 1 і стовпчика 2, а закінчувався в рядку 3 і стовпчику 2: 

.item4 {
  grid-area: 1 / 2 / 3 / 2;
}

Іменування елементів за допомогою grid-area

Властивість grid-area також можна використовувати для присвоєння імен елементам сітки.

На названі елементи сітки можна посилатися за допомогою властивості grid-template-areas контейнера сітки. 

.item1 { grid-area: header; }
.item2 { grid-area: menu; }
.item3 { grid-area: main; }
.item4 { grid-area: right; }
.item5 { grid-area: footer; }

.grid-container {
  grid-template-areas:
    'header header header header header header'
    'menu main main main main right'
    'menu footer footer footer footer footer';
}
