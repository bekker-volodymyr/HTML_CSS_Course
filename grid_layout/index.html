Grid Layout: Вивчення та Практика

Заняття 1: Основи та ключові властивості Grid Layout

Що таке Grid Layout?

Grid Layout — це потужний інструмент CSS для створення гнучких та структурованих макетів веб-сторінок за допомогою поділу на рядки та колонки. На відміну від Flexbox, який краще підходить для вирівнювання елементів по одному напряму, Grid дозволяє працювати з двома вимірами одночасно.

Основні поняття

  • Грід-контейнер: батьківський елемент, на якому застосовується властивість display: grid;.
  • Грід-елементи: дочірні елементи грід-контейнера, які автоматично розташовуються в рядках і колонках.
  • Рядки та колонки: основні блоки побудови гріда.
  • Трек: рядок або колонка гріда.
  • Гапи: проміжки між треками.

1. Створення грід-контейнера

Для початку необхідно задати контейнеру властивість display: grid;. Наприклад:

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: auto;
  gap: 10px;
}

Цей код створює грід із трьома колонками рівної ширини.

2. Властивість grid-template-columns та grid-template-rows

Визначає кількість та розміри колонок та рядків.

.example-grid {
  display: grid;
  grid-template-columns: 100px 200px 1fr;
  grid-template-rows: 50px auto;
}

Тут перша колонка має ширину 100px, друга — 200px, а третя займає весь залишковий простір.

3. Властивість gap

Визначає відступи між рядками та колонками.

.grid-gap-example {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 20px;
}

Цей код встановлює відступи у 20px між усіма грід-елементами.

Заняття 2: Динамічні розміри та вирівнювання

1. Властивість minmax()

Задає діапазон розмірів для рядків або колонок.

.grid-minmax-example {
  display: grid;
  grid-template-columns: repeat(3, minmax(100px, 1fr));
}

Колонки матимуть мінімальну ширину 100px і можуть розтягуватися, займаючи залишковий простір.

2. Властивості auto-fit та auto-fill

Ці значення дозволяють динамічно заповнювати грід-сітку колонками або рядками.

.grid-auto-fit-example {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 15px;
}

При використанні auto-fit колонки будуть підлаштовуватись під доступний простір.

3. Вирівнювання елементів у грід-контейнері

Властивості для вирівнювання:

  • place-items: одночасне вирівнювання по горизонталі та вертикалі.
  • justify-items: вирівнювання по горизонталі.
  • align-items: вирівнювання по вертикалі.
.grid-alignment-example {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  place-items: center;
  gap: 10px;
}
