Підключення Bootstrap

Bootstrap — це безкоштовний фронтенд-фреймворк для швидшої та зручнішої розробки вебсайтів.

Bootstrap включає шаблони дизайну на основі HTML та CSS для типографії, форм, кнопок, таблиць, навігації, модальних вікон, каруселей зображень та багатьох інших елементів, а також необов'язкові JavaScript плагіни.

Bootstrap також дає змогу легко створювати адаптивні дизайни.

Де отримати Bootstrap 5?

Є два способи почати використовувати Bootstrap 5 на своєму вебсайті.

Ви можете:

  • Включити Bootstrap 5 з CDN
  • Завантажити Bootstrap 5 з getbootstrap.com

Bootstrap 5 CDN

Якщо ви не хочете завантажувати та хостити Bootstrap 5 на своєму сервері, ви можете включити його з CDN (Content Delivery Network).

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Колірна схема Bootstrap

Bootstrap пропонує використовувати десять різних класів для зміни кольору тексту або фону елемента. Назви класів пов'язані не з кольором, а з контекстом елемента. Виділяють наступні класи:

  • primary — основний (первинний) колір
  • secondary — допоміжний (вторинний) колір
  • success — колір вдалого виконання дії
  • danger — колір невдалого виконання дії
  • warning — колір попередження (акценту)
  • info — колір інформаційних блоків
  • light — світлий (rgb(248, 249, 250))
  • dark — темний (rgb(52, 58, 64))
  • white — білий
  • transparent — прозорий

Для того, щоб застосувати колір для тексту, слід перед префіксом написати «text-», а для завдання фонового кольору — клас «bg-».

<div class="bg-dark text-white">bg-dark text-white</div>
<div class="bg-light text-dark">bg-light text-dark</div>
<div class="bg-info text-light">bg-info text-light</div>

Grid система

Фреймворк Bootstrap пропонує гнучку систему побудови сіткової структури, з використанням якої можна порівняно легко та швидко отримати адаптивну сторінку.

Bootstrap використовує п'ять типорозмірів ширини пристрою для формування сітки. Кожному пристрою відповідає свій префікс класу.

Пристрій Ширина пристрою Префікс класу
Мобільні телефони <576px Без префікса
Мобільні телефони ≥576px -sm
Планшети ≥768px -md
Монітори ≥992px -lg
Широкоформатні монітори ≥1200px -xl

Основним елементом структури сітки є контейнер, який призначений для розміщення та/або центрування елементів всередині нього.

Bootstrap пропонує використовувати три різних класи для контейнера:

  • container — встановлює фіксовану ширину для кожного пристрою.
  • container-fluid — встановлює ширину, що дорівнює 100% ширині батьківського елемента.
  • container-{префікс класу, що визначає пристрій} — встановлює ширину 100% для пристроїв, менших за вказаний у префіксі, та фіксовану ширину для зазначеного пристрою.

Всередині контейнера знаходяться рядки (елементи з класом «row»), кожен з яких містить стовпці (елементи з класом «col»).

Класи стовпців мають число, що вказує на кількість стовпців у рядку. Максимальна кількість стовпців у рядку — 12. Ширина стовпця ніколи не фіксується, оскільки задається у відсотках від ширини батьківського елемента-рядка.

col bg-dark text-white
col bg-primary text-white
col bg-success text-white
col bg-warning text-white
col bg-info text-white

На мобільних пристроях стовпці займатимуть усю ширину контейнера, на планшетах — половину, на моніторах і широкоформатних моніторах — 1/3 частину ширини контейнера.

Утиліти

Утилітами називають допоміжні функції, які виконують певні завдання і полегшують верстання. Фреймворк Bootstrap пропонує 20 різних утиліт. Розглянемо застосування найбільш використовуваних із них.

Рамка

Щоб задати межі елементу, достатньо прописати один з класів: border, border-top, border-right, border-bottom, border-left.

Щоб додати колір до рамки, достатньо додати клас «border border-*», в якому зірочка — це назва потрібного кольору класу.

Для того, щоб закруглити кути елемента, пропишіть один з класів: rounded, rounded-top, rounded-bottom, rounded-left, rounded-right. Для більшого або меншого радіусу необхідно додати в кінець префікс -lg або -sm до зазначених класів.

Для отримання кола потрібно використовувати клас rounded-circle для квадратних елементів. А для отримання напівкруглих меж у прямокутних елементів скористайтеся класом rounded-pill.

border border-primary rounded
border-bottom border-success
border border-bottom-0 border-warning
border border-dark rounded-pill 
<div class="container">
    <div class="row row-cols-4">
        <div class="col border border-primary rounded">
            border border-primary rounded
        </div>
        <div class="col border-bottom border-success">
            border-bottom border-success
        </div>
        <div class="col border border-bottom-0 border-warning">
            border border-bottom-0 border-warning
        </div>
        <div class="col border border-dark rounded-pill">
            border border-dark rounded-pill
        </div>
    </div>
</div>

Тіні

Додати або видалити тінь можна за допомогою класів:

  • shadow-none — тінь відсутня;
  • shadow-sm — невелика тінь;
  • shadow — звичайна тінь;
  • shadow-lg — велика тінь.
<div class="container">
    <div class="row row-cols-4">
        <div class="col shadow-none">shadow-none</div>
        <div class="col shadow-sm">shadow-sm</div>
        <div class="col shadow">shadow</div>
        <div class="col shadow-lg">shadow-lg</div>
    </div>
</div>

Поля та відступи

Bootstrap пропонує використовувати класи для завдання адаптивних зовнішніх і внутрішніх відступів.

Класи будуються за наступним форматом: [властивість][сторона]-[префікс класу, що визначає пристрій]-[розмір].

Властивість може бути:

  • m — margin (зовнішній відступ)
  • p — padding (внутрішній відступ)

Сторона або сторони визначається наступними префіксами:

  • t — верхня
  • b — нижня
  • l — ліва
  • r — права
  • x — ліва і права
  • y — верхня і нижня
  • порожньо — всі 4 сторони елемента

Розмір визначається одним із наступних префіксів:

  • 0 — видалення полів або відступів, встановлюється значення 0
  • 1 — встановлюється значення 0.25
  • 2 — встановлюється значення 0.5
  • 3 — встановлюється значення 1
  • 4 — встановлюється значення 1.5
  • 5 — встановлюється значення 3
  • auto — встановлює значення auto для зовнішніх відступів

Розглянемо приклад. Створимо 4 стовпчики з різними внутрішніми відступами зліва для всіх видів пристроїв і з верхнім або нижнім внутрішнім відступом для пристроїв, починаючи з планшета. Також додамо зовнішній верхній відступ для пристроїв, починаючи з планшета.

<div class="container-fluid">
    <div class="row row-cols-4">
        <div class="col bg-info pl-5 pt-md-2 mt-md-3">
            pl-5 pt-md-2 mt-md-3
        </div>
        <div class="col bg-warning pl-4 pb-md-2 mt-md-3">
            pl-4 pb-md-2 mt-md-3
        </div>
        <div class="col bg-light pl-3 pt-md-2 mt-md-3">
            pl-3 pt-md-2 mt-md-3
        </div>
        <div class="col bg-success pl-2 pb-md-2 mt-md-3">
            pl-2 pb-md-2 mt-md-3
        </div>
    </div>
</div>

Відображення

Щоб показати або приховати елементи на різних пристроях, потрібно скористатися класом, який утворюється за таким форматом: d-[префікс класу, що визначає пристрій]-[значення].

Значенням може бути одне із значень властивості display:

  • none;
  • inline;
  • inline-block;
  • block;
  • table;
  • table-cell;
  • table-row;
  • flex;
  • inline-flex;

Розглянемо на прикладі створення рядка із 5 стовпців:

  • Перший стовпчик буде прихованим тільки на найменшому пристрої, а на решті пристроїв відображатиметься.
  • Другий стовпчик буде прихованим тільки при відображенні на мобільних пристроях.
  • Третій стовпчик відображатиметься на усіх пристроях, крім планшетів.
  • Четвертий стовпчик буде прихованим на усіх пристроях, крім мобільних телефонів.
  • П'ятий стовпчик буде прихованим на усіх пристроях, крім планшетів.
<div class="container">
    <div class="row">
        <div class="col border d-none d-sm-block">
            1 d-none d-sm-block
        </div>
        <div class="col border d-sm-none d-md-block">
            2 d-sm-none d-md-block
        </div>
        <div class="col border d-md-none d-lg-block">
            3 d-md-none d-lg-block
        </div>
        <div class="col border d-none d-sm-block d-md-none">
            4 d-none d-sm-block d-md-none
        </div>
        <div class="col border d-none d-md-block d-lg-none">
            5 d-none d-md-block d-lg-none
        </div>
    </div>
</div>

Зображення та відео

Зображення є невід'ємною частиною будь-якого сайту. Фреймворк Bootstrap пропонує використовувати наступні класи для зображень.

Клас Опис
img-fluid Задає зображенню властивості «max-width: 100%;» та «height: auto;».
img-thumbnail Використовується для додання зображенню закругленої межі в 1px.
float-left Застосовується для вирівнювання зображення за лівим краєм.
float-right Застосовується для вирівнювання зображення за правим краєм.
mx-auto d-block Застосовується для вирівнювання зображення по центру. Для цього потрібно перетворити зображення на блоковий елемент використовуючи клас d-block.

Для вбудованих елементів, таких, як <iframe>, <embed>, <video> та <object>, необхідно використовувати блоковий елемент-обгортку з класом «ratio». Для вбудованих елементів задається відповідне співвідношення сторін.

Також можливо налаштувати співвідношення сторін за допомогою класів-модифікаторів:

  • ratio-21x9
  • ratio-16x9
  • ratio-4x3
  • ratio-1x1
