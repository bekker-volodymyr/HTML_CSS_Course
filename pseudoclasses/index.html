Псевдоелементи

Псевдоелементи CSS використовуються для стилізації певних частин елемента.

Наприклад, їх можна використовувати для:

  • стилізації першої літери або рядка елемента (first-letter/first-line)
  • додавання контенту до або після елемену (before/after)
  • стилізації маркерів елементів списку
  • стилізації області перегляду (viewbox) за діалоговим вікном

Псевдоелемент ::first-line

За назвою цього псевдоелемента можна здогадатися, що призначений він для форматування першого рядка блокових елементів. Для оформлення першого рядка можна використовувати не всі CSS-властивості, а тільки ті, що наведені в списку нижче:

  • Властивості групи font (font-family, font-size, font-style, font-weight, font-variant та ін.)
  • color
  • Властивості групи background
  • word-spacing
  • letter-spacing
  • text-decoration, text-decoration-color, text-decoration-line, text-decoration-style
  • vertical-align
  • text-shadow
  • text-transform
  • line-height
  • clear

Псевдоелемент ::first-letter

За назвою цього псевдоелемента також можна зрозуміти, що його призначення — оформити першу літеру блокового елемента.

Властивості доступні для псевдоелементу ::first-letter

  • font (всі підвластивості: font-family, font-size, font-style, font-variant, font-weight)
  • color
  • background (всі підвластивості)
  • margin
  • padding
  • border (всі підвластивості)
  • text-decoration
  • text-transform
  • line-height
  • float
  • vertical-align

Псевдокласи

Псевдоклас використовується для визначення особливого стану елемента.

Наприклад, його можна використовувати для:

  • стилізації елемента, коли користувач наводить на нього курсор
  • стилізації відвіданих і невідвіданих посилань по-різному
  • стилізації елемента, коли він отримує фокус
  • стилізації полів форми залежно від їхнього стану (валідні, невалідні, обов’язкові, необов’язкові)

:first-child | :last-child

:first-child та :last-child — це псевдокласи, які використовуються для вибору першого або останнього дочірнього елемента всередині батьківського контейнера.

:first-child

  • Застосовується до першого дочірнього елемента в батьківському контейнері.
  • Спрацьовує тільки якщо елемент дійсно є першим серед усіх сусідів.

:last-child

  • Застосовується до останнього дочірнього елемента в батьківському контейнері.

Важливо: Якщо всередині батьківського контейнера є інші елементи (наприклад, h1, span), то :first-child і :last-child спрацюють тільки якщо відповідний елемент є справді першим або останнім серед усіх дочірніх елементів. 

p:first-child {
    font-family: 'Georgia', serif;
    font-size: 1.2rem;
    background-color: #e0f7fa;
    padding: 10px;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
p:last-child {
    font-family: 'Verdana', sans-serif;
    font-size: 1.4rem;
    background-color: #ffe0b2;
    padding: 10px;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
Приклад

:nth-child()

Для того, щоб відформатувати якийсь певний елемент, необхідно використовувати псевдоклас :nth-child(n). Замість n потрібно поставити будь-яку цифру. 

p:nth-child(2) {
    font-family: 'Georgia', serif;
    font-size: 1.2rem;
    background-color: #e0f7fa;
    padding: 10px;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
Приклад

Якщо необхідно задати форматування для кожного 3-го абзацу, варто використовувати псевдоклас :nth-child(3n). Замість цифри 3 може бути будь-яка цифра або вираз, наприклад, 4n-1, 3n+1 тощо 

p:nth-child(3n) {
    background-color: #e0f7fa;
    font-size: 1.2rem;
    padding: 10px;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
Приклад

Досить часто буває потріба у форматуванні парних або непарних елементів. Для цієї мети можна використовувати, наприклад, псевдокласс :nth-child(2n) або :nth-child(2n-1). Крім того, існують спеціальні ключові слова для парних (even) і непарних (odd) елементів. 

.box:nth-child(odd) {
    background-color: #f99;
}
.box:nth-child(even) {
    background-color: #f8f48c;
}
Приклад

:nth-last-child

Псевдоклас :nth-last-child(n) призначений для форматування елементів, відлік яких ведеться не з початку їх появи в коді, тобто з першого елемента, а навпаки — з кінця, тобто з останнього елемента.

В дужках замість n можна записувати число, вираз (2n, 3n-1, 4n+1 і т.п.), а також ключові слова odd (непарні числа) або even (парні числа). 

селектор:nth-last-child( число | вираз | odd | even) {
    властивості;
}

:only-child

Псевдоклас :only-child призначений для форматування елементів, які є єдиним дочірнім для свого батьківського тега. Тобто він буде застосований тільки в тому випадку, якщо в батьківському елементі для зазначеного селектора відсутні інші елементи такого ж типу.

Приклад

:nth-of-type()

Псевдоклас :nth-of-type(n) використовується для вибору елементів певного типу (наприклад, p, div) в межах батьківського контейнера, враховуючи їхній порядок серед елементів одного типу. 

p:nth-of-type(2) {
    font-family: 'Georgia', serif;
    font-size: 1.2rem;
    background-color: #e0f7fa;
    padding: 10px;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
Приклад

Замість цифри 2 можна використовувати вирази, наприклад, 3n, 4n-1 тощо, щоб задати певні елементи на основі їхнього порядку серед елементів одного типу. 

p:nth-of-type(3n) {
    background-color: #e0f7fa;
    font-size: 1.2rem;
    padding: 10px;
    border-radius: 8px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
Приклад

Аналогічно, можна використовувати ключові слова odd (непарні елементи) і even (парні елементи) для вибору відповідних елементів. 

.box:nth-of-type(odd) {
    background-color: #f99;
}
.box:nth-of-type(even) {
    background-color: #f8f48c;
}
Приклад

:nth-last-of-type

Псевдоклас :nth-last-of-type(n) працює аналогічно до :nth-of-type, але відлік елементів ведеться з кінця, тобто з останнього елемента того ж типу. 

селектор:nth-last-of-type(число | вираз | odd | even) {
    властивості;
}
