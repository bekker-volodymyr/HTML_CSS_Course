Заняття 2: Розширення знань з HTML та введення в CSS

Розминка та коротке повторення

  • Базова структура HTML-документа.
  • Основні теги: <html>, <head>, <body>, <title>, <meta charset>, <h1>-<h6>, <p>, <br>.

Розширення знань з HTML

Додаткові теги

Розглянемо інші важливі теги:

  • Списки: ненумеровані (<ul>) та нумеровані (<ol>) списки, елементи списків (<li>).
  • Блочний контейнер: <div> – використовується для групування контенту.

Розширені текстові теги

Деякі важливі текстові теги:

  • <b>: жирний текст, який не має особливого семантичного значення.
  • <strong>: важливий текст, який підкреслює смислову значущість.
  • <i>: курсивний текст, зазвичай використовується для акцентів.
  • <em>: курсивний текст із акцентом на змісті.

Блочні та інлайн елементи

Блочні елементи займають всю ширину контейнера та починають новий рядок. Прикладом є <div>, <p>, <h1>.

Інлайн елементи займають лише стільки простору, скільки необхідно їхньому контенту, і не починають новий рядок. Прикладом є <span>, <a>, <em>.

Аналогія: уявіть блок як коробку з текстом, яка займає всю полицю, а інлайн — як книжку, яку можна поставити поруч із іншими книжками на тій же полиці.

Графічна схема:

Уявіть таку структуру:

| [Блоковий елемент: Заголовок ]        |
| [Блоковий елемент: абзац]             |
| [Інлайн елементи: слово1 слово2 слово3]

Атрибути тегів

Важливість використання атрибутів:

  • ID та class: для унікального та групового стилювання.
  • Універсальні атрибути: style, data-* – для додаткової інформації або стилізації.

Введення в CSS

CSS (Cascading Style Sheets) використовується для оформлення HTML-документів. Розглянемо:

  • Основні стильові властивості: кольори, шрифти, розміри, відступи.
  • Способи підключення CSS:
    • Inline стилі: використання атрибуту style в HTML-тегах.
    • Зовнішній файл: створення окремого файлу CSS та підключення його через тег <link> в <head>.

Практика

Практичні завдання допоможуть закріпити матеріал:

  • Створити блок із заголовком, абзацом і списком, застосувавши стилі.
  • Додати інлайн-елементи та застосувати до них різні стилі.
