Позиціонування

За допомогою CSS-властивості position ми можемо налаштувати логіку за якою елемент розміщається на сторінці.

У нас є 5 різних варіантів:

static
звичайне позиціонування елементу в потоці сторінки (за замовчуванням)
relative
позиціонування відносно статичної позиції
fixed
розташування відносно вікна перегляду (viewport)
absolute
розташування відносно батьківського елемента (але не статичного)
sticky
розташування відносно прокрутки

Після вказання типу позиціонування ми можемо налаштувати позицію елемента властивостями:

  • left
  • top
  • bottom
  • right

Ці властивості представляють відступ від точки позиціонування.

Поки не вказаний тип позиціонування ці властивості не будуть мати ефекту.

Властивості left top bottom right - це відстань. Задається за допомгою абсолютних (px,cm) або відносних (%, vw, vh) величин.

Приклад з усіма типами позиціонування

Значення position за замовченням для всіх елементів є static. Це означає, що ніякого особливого позиціонування не застосовується. Позиція розраховується згідно потоку сторінки.

Приклад статичного позиціонування

Відносне позиціонування

Для відносного позиціонування використовуємо значення relative властивості position.

При такому позиціонуванні місце елементу буде розраховуватись від його звичайної позиції.

Наприклад:

div {
    position: relative;
    left: 35px;
}

В цьому випадку елемент зміститься на 35 пікселів ліворуч від своєї звичайної позиції.

Встановіть значення left top bottom або right щоб змістити елемент відносно його звичайної позиції у відповідному напрямку.

Таке позиціонування не вплине на розміщення інших елемнтів. Вони не заповнять утворений простір і можуть накладатись один на одний.

Приклад відносного позиціонування.

Фіксоване позиціонування

Елемент із position: fixed; розташовується відносно вікна перегляду (viewport), що означає, що він завжди залишається на одному місці, навіть якщо сторінка прокручується.

Властивості top right bottom і left використовуються для визначення його положення - це відступи від відповідних сторін вікна перегляду (viewport).

Фіксований елемент не залишає порожнього місця на сторінці там, де він зазвичай знаходився б.

Приклад фіксованого позиціонування.

Встановіть значення position на fixed, а значення top на 0, щоб створити елемент, що завжди буде згори екрану: 

div {
    position: fixed;
    top: 0;
}

Абсолютне позиціонування

Елемент із position: absolute; розташовується відносно наближчого батьківського елемнту з заданим (не статичним) позиціонуванням (замість розташування відносно вікна перегляду, як у fixed).

Однак, якщо абсолютний елемент не має позиціонованих предків, він використовує body як точку відліку та переміщується разом із прокруткою сторінки (аналогічно fixed).

Примітка: Абсолютно позиціоновані елементи вилучаються з нормального потоку документа і можуть перекривати інші елементи.

Приклад абсолютного позиціонування

Липке позиціонування

Елемент із position: sticky; розташовується залежно від положення прокрутки користувача.

Sticky-елемент перемикається між relative і fixed, залежно від прокрутки. Спочатку він поводиться як відносно позиціонований (relative), але коли досягає вказаного зміщення у вікні перегляду, він "прилипає" і фіксується на місці (як fixed).

Щоб змусити елемент "прилипнути" до верхнього краю вікна задайте наступні стилі:

div {
    position: sticky;
    top: 0;
}
Приклад липкого позиціонування

Використання z-index

Коли елементи позиціонуються, вони можуть перекривати інші елементи.

Властивість z-index визначає порядок накладання елементів (який елемент має бути розташований попереду або позаду інших).

Примітка: Властивість z-index працює тільки для позиціонованих елементів (position: absolute;, position: relative;, position: fixed;, position: sticky;) та flex-елементів (тобто елементів, які є безпосередніми нащадками елемента з display: flex;).

Якщо два позиціоновані елементи перекривають один одного без вказаного z-index, то елемент, який визначений останнім у HTML-коді, буде відображатися поверх інших.

Приклад використання z-index.

Відображення

Властивість display використовується для вказання, як елемент буде відображатись на сторінці.

Значень властивості display багато. Розглянемо основні:

inline
відображає елемент як рядковий (елемент займає стільки місця, скільки його контент, задання margin, width, height недоступні)
block
відображає елемент як блоковий
flex
перетворює елемент на flex-контейнер
grid
перетворює елемент на grid-контейнер
none
прибирає елемент зі сторінки

Випадаючий елемент

Розглянемо наступний приклад:

Випадаючий елемент

Для реалізації такого прикладу нам знадобляться властивості position та display. 

.hover-elem {
    position: relative;
    width: 150px;
    height: 50px;
    margin: auto;
    background-color: brown;
    text-align: center;
    padding-top: 25px;
}

.hidden-elem {
    display: none;
    position: relative;
    width: 100px;
    height: 70px;
    background-color: burlywood;
    margin: auto;
    top: 30px;
}

.hover-elem:hover .hidden-elem {
    display: block;
}

Важливо, щоб випадаючий (прихований) елемент був дочірнім елементу, якому налаштовуємо ефект hover. 

<div class="hover-elem">Наведи на мене!
    <div class="hidden-elem">Я прихований елемент!</div>
</div>
