Тег table

Тег table дозволяє організувати інформацію в якості рядків та стовпчиків.

Приклад:

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Тег td

Тег td визначеє комірку таблиці.

td - скорочення від table data.

Тег td може містити будь який елемент всередині - тексти, зображення тощо.

<table>
    <tr>
        <td>Emil</td>
        <td>Tobias</td>
        <td>Linus</td>
    </tr>
</table>

Тег tr

Тег tr представляє рядок в таблиці.

Ви можете мати скільки завгодно рядків у таблиці; просто переконайтеся, що кількість клітинок однакова в кожному рядку.

Заголовки таблиці

Іноді потрібно, щоб клітинки були клітинками заголовків таблиці. У таких випадках використовуйте тег <th> замість тегу <td>

<tr>  
    <th>Person 1</th>  
    <th>Person 2</th>  
    <th>Person 3</th>  
</tr>

Стилізація таблиць

Ви можете стилізувати таблицю за допомгою відповідних селекторів: table, th, tr, td.

table, th,td {
    border: 1px solid black;
    border-collapse: collapse;
}

Заголовки таблиць

Ви можете задати розміщення тексту в заголовку за допомогою text-align.

Ви можете створити заголовок, що розтягується на декілька стовпчиків використовуючи атрибут colspan.

За допомогою тегу caption до таблиці можна додати загальний заголовок

Наприклад:

Таблиця якихось людей
Name Age
Jill Smith 50
Eve Jackson 94

Відступи між комірками

Відстань між комірками (cell spacing) — це простір між кожною коміркою таблиці.

За замовчуванням ця відстань встановлена на 2 пікселі.

Щоб змінити відстань між комірками таблиці, використовуйте CSS-властивість border-spacing для елемента <table>.

table {
    border-spacing: 30px;
}

colspan та rowspan

Використовуючи атрибути colspan та rowspan ви можете змусити комірки розтягнутись на декілька стовпчиків чи рядків відповідно.

Name Age
Jill Smith 43
Eve Jackson 57
Name Jill
Phone 555-1234
555-8745
