Що таке flexbox?

Flexbox - це скорочення для Flexible Box Layout module.

Flexbox - це спосіб розміщення елементів на сторінці в рядки та стовпчики.

Flexbox дозволяє легко розробляти гнучкий адаптивний дизайн без використання float чи position.

Уявіть, що ви створюєте фотогалерею. У вас є декілька фотографій різного розміру, і вам потрібно рівномірно розташувати їх по горизонталі так, щоб між ними був однаковий простір та всі фото були гармонійно вирівняні по центру.

Без Flexbox вам доведеться вручну задавати відступи та вирівнювання, що може бути складно і не завжди виглядає естетично.

Flexbox складається з:

  • Flex-контейнер - батьківський контейнер (display: flex).
  • Flex-елемент - всі елементи всередині батьківського flex-контейнеру.

Flexbox vs. Grid

Flexbox - одно-направлена розмітка - рядки АБО колонки.

Grid - дво-направлена розмітка - рядки ТА колонки.

Приклад без Flexbox

<div style="border: 1px solid #000; padding: 10px;">
    <img src="resources/photo1.png" alt="Фото 1" style="width: 200px; margin-right: 10px;">
    <img src="resources/photo2.png" alt="Фото 2" style="width: 200px; margin-right: 10px;">
    <img src="resources/photo3.png" alt="Фото 3" style="width: 200px;">
</div>
  • Фото можуть виглядати нерівномірно через різні розміри та жорстко задані відступи.
  • При зміні розміру вікна браузера розташування може порушуватися.

Приклад з Flexbox

<div style="display: flex; justify-content: space-around; align-items: center;">
    <img src="resources/photo1.png" alt="Фото 1" style="width: 200px;">
    <img src="resources/photo2.png" alt="Фото 2" style="width: 200px;">
    <img src="resources/photo3.png" alt="Фото 3" style="width: 200px;">
</div>
  • display: flex; – робить контейнер гнучким.
  • justify-content: space-around; – автоматично розподіляє простір між фото.
  • align-items: center; – вирівнює фото по вертикалі.

Flex-контейнер

Елемент стає flex-контейнером, якщо йому встановити властивість display на значення flex. 

.flex-container {
    display: flex;
}

Flex-контейнеру доступні наступні властивості:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
  • gap

flex-direction

Властивість flex-direction визначає напрямок розміщення елементів - рядок чи стовпчик.

Вісь, зазначена у властивості flex-direction називається основною. Інша, перпендикулярна до основної називається поперечною.

Доступні значення:

  • row(за замовчуванням)
  • column
  • row-reverse
  • column-reverse
.flex-container {
    display: flex;
    flex-direction: row | column | row-reverse | column-reverse;
}

flex-wrap

Властивість flex-wrap визначає, чи повинні елементи флекс-контейнера переноситися на новий рядок (або стовпчик), якщо для них не вистачає місця.

Доступні значення

  • nowrap (за замовчуванням)
  • wrap
  • wrap-reverse

justify-content

Властивість justify-content вирівнює флекс-елементи по основній осі. Основні значення:

  • flex-start – елементи вирівнюються на початку контейнера.
  • flex-end – елементи вирівнюються в кінці контейнера.
  • center – елементи вирівнюються по центру.
  • space-between – елементи розподіляються з рівномірними відступами, перший елемент вирівнюється з початком, останній – з кінцем.
  • space-around – елементи мають рівномірні відступи з обох боків.
  • space-evenly - простір рівномірно розподіляється між елементами.

align-items

Властивість align-items вирівнює флекс-елементи по поперечній осі. Основні значення:

  • flex-start – елементи вирівнюються зверху контейнера.
  • flex-end – елементи вирівнюються знизу.
  • center – елементи вирівнюються по центру.
  • baseline – вирівнювання за базовою лінією тексту.
  • stretch – елементи розтягуються по висоті контейнера (за замовчуванням).

align-content

Властивість align-content використовується для вирівнювання рядків(стовпчиків) флекс-контейнера.

Властивість align-content схожа на align-items, але замість вирівнювання елементів флекс-контейнера, вона вирівнює самі флекс-рядки(стовпчики).

Властивість align-content може мати одне з наступних значень:

  • center
  • stretch
  • flex-start
  • flex-end
  • space-around
  • space-between
  • space-evenly

Flex-елементи

Всі дочірні елементи flex-контейнеру стають flex-елементами автоматично.

Властивості, доступні flex-елементам:

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

order

Властивість order визначає порядок елементу всередині контейнеру. 

<div class="flex-container">
    <div style="order: 3">1</div>
    <div style="order: 2">2</div>
    <div style="order: 4">3</div>
    <div style="order: 1">4</div>
</div>

flex-grow

Властивість flex-grow визначає, наскільки елемент може розтягуватись у порівнянні з іншими елементами.

Якщо для одного з елементів задано значення flex-grow: 2;, а для інших flex-grow: 1;, цей елемент займе вдвічі більше простору.

flex-grow

1 (grow: 1)
2 (grow: 2)
3 (grow: 1)

flex-shrink

Властивість flex-shrink визначає, наскільки елемент може зменшуватись у розмірах у порівнянні з іншими елементами при нестачі простору.

Якщо для одного з елементів задано значення flex-shrink: 0;, він не буде зменшуватись.

Приклад

flex-basis

Властивість flex-basis задає початковий розмір елемента перед тим, як буде застосовано властивості flex-grow і flex-shrink.

Вона може бути задана у пікселях, відсотках або автоматично (за значенням auto).

flex-basis

1 (basis: 50px)
2 (basis: 100px)
3 (basis: 150px)

align-self

Властивість align-self визначає вирівнювання вибраного елемента всередині флекс-контейнера.

Властивість align-self переважає за замовчуванням вирівнювання, яке встановлюється властивістю align-items контейнера.
