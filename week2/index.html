Тег <pre>

Тег <pre> використовується для відображення тексту у фіксованому форматі з урахуванням пробілів та нових рядків.

<pre>
    Це приклад преформатованого тексту.
    Всі пробіли та рядки збережені.
</pre>

Інший приклад преформатованого тексту:

My Bonnie lies over the ocean.

My Bonnie lies over the sea.

My Bonnie lies over the ocean.

Oh, bring back my Bonnie to me.

Цитування

Для створення цитати з іншого джерела використовується тег <blockquote>

<blockquote cite="http://www.worldwildlife.org/who/index.html">
    For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, 
    WWF works in nearly 100 countries. At every level, we collaborate with people around the world to develop and 
    deliver innovative solutions that protect communities, wildlife, and the places in which they live.
</blockquote>

Для коротких цитат можна використовувати тег <q>:

<p>WWF's goal is to: <q>Build a future where people live in harmony with nature.</q></p>

Результат:

WWF's goal is to: Build a future where people live in harmony with nature.

HTML-тег <abbr> визначає абревіатуру або акронім, такі як "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Позначення абревіатур може надати корисну інформацію браузерам, системам перекладу та пошуковим системам. Порада: Використовуйте глобальний атрибут title, щоб відобразити опис абревіатури/акроніма при наведенні курсору на елемент.

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

Результат:

The WHO was founded in 1948.

Селектори

Ми можемо розділити CSS-селектори на п’ять категорій:

  • Прості селектори (вибирають елементи за ім’ям, id, класом).
  • Комбінаторні селектори (вибирають елементи на основі певного зв’язку між ними).
  • Селектори псевдокласів (вибирають елементи залежно від їхнього стану).
  • Селектори псевдоелементів (вибирають і стилізують частину елемента).
  • Селектори атрибутів (вибирають елементи на основі атрибута або його значення).

Селектор елементу

Обирає HTML елемент з відповідною назвою. 

p {
    text-align: center;
    color: red;
}

Селектор id

Селектор id використовує атрибут id HTML-елемента для вибору конкретного елемента. id елемента є унікальним на сторінці, тому селектор id використовується для вибору лише одного унікального елемента! Щоб вибрати елемент із певним id, напишіть символ решітки (#), а потім ім'я id елемента.

#para1 {
    text-align: center;
    color: red;
}

Селектор класу

Селектор класу вибирає HTML-елементи з певним атрибутом class. Щоб вибрати елементи з конкретним класом, напишіть крапку (.), а потім ім'я класу. 

.center {
    text-align: center;
    color: red;
}

Ви також можете вказати, що лише певні HTML-елементи повинні бути під впливом класу. 

p.center {
    text-align: center;
    color: red;
}

HTML елементи можуть мати декілька класів. Всі класи вказуємо через пробіл.

<p class="center large">This paragraph refers to two classes.</p>

Універсальний селектор

Універсальний селектор використовується для задання стилів всіх HTML елементів.

* {
    text-align: center;
    color: blue;
}

Груповий селектор

Груповий селектор використвується коли декілька елементів мають однакову стилізацію:

h1, h2, p {
    text-align: center;
    color: red;
}

Зовнішні стилі

Ми вже використовували два способи підключення стилів: inline (до елементу) та сторінковий (тег style)

Для великих сайтів зазвичай використовується третій спосіб: підключення зовнішнього файлу.

Спочатку необхідно створити файл з розширенням .css

Щоб підключити створений файл стилів до HTML документу використовуємо тег link всередині тегу head

<link rel="stylesheet" href="style.css">

Коментарі в CSS документі

Коментарі у CSS створюються за допомогою символів /* (початок коментарю) */ (кінець коментарю). 

/* This is a single-line comment */
p {
    color: red;
}

Вирівнювання тексту

Для вирівнювання тексту всередині блоку використвується властивість text-align

Текст вирівняний ліворуч (text-align: left).

Текст вирівняний по центру (text-align: center).

Текст вирівняний праворуч (text-align: right).

Текст вирівняний по ширині (text-align: justify). Це дозволяє тексту зайняти всю ширину блоку, зокрема додаючи проміжки між словами.

Кольори

Для налаштування кольору тексту використовується властивість color

Для налаштування кольору фону елементу використовується властивість background-color

Варіанти вказування кольорів

  • color: blue; /* назва кольору англійською мовою */    

  • color: #ff0000; /* 16-річне значення кольору, що використовує цифри від 0 до 9 та літери від a до f */

  • color: #f00; /* усічене 16-річне значення для кольорів, у яких однакові пари цифр */

  • color: rgb (255, 0, 0); /* значення кольору у системіrgb. По кожному каналу r - red, g - green, b - blue значення кольору змінюються від 0 до 255 одиниць*/

  • color: rgba(255, 0, 0, .5); /* значення кольору системіrgb+ альфа-канал, тобто, можливість задати прозорість для кольору від 0 до 1*/

  • color: hsl(0, 100%, 50%); /* значення кольору у системіhsl- hue -saturation - lightness */

  • color: hsla(0, 100%, 50%, .5); /* значення кольору у системіhsl+ альфа-канал, тобто, можливість задати прозорість для кольору від 0 до 1*/

Шрифти

Загальні сімейства шрифтів

У CSS існує п’ять загальних сімейств шрифтів:

  • Serif – шрифти з невеликими засічками на краях кожної літери. Вони створюють відчуття формальності та елегантності.
  • Sans-serif – шрифти без засічок, з чистими лініями. Вони виглядають сучасно та мінімалістично.
  • Monospace – у цих шрифтах усі літери мають однакову фіксовану ширину, що створює механічний вигляд.
  • Cursive – шрифти, що імітують людський почерк.
  • Fantasy – декоративні або ігрові шрифти.

Усі різні назви шрифтів належать до одного з цих загальних сімейств шрифтів.

Властивість font-family в CSS

У CSS ми використовуємо властивість font-family, щоб вказати шрифт для тексту.

Примітка: Якщо назва шрифту складається з кількох слів, її потрібно брати в лапки, наприклад: "Times New Roman".

Порада: Властивість font-family повинна містити кілька назв шрифтів як систему резервних варіантів ("fallback"), щоб забезпечити максимальну сумісність між браузерами та операційними системами. Починайте з бажаного шрифту і завершуйте загальним сімейством (щоб браузер міг вибрати схожий шрифт, якщо інші недоступні). Назви шрифтів потрібно розділяти комами. 

.p1 {
    font-family: "Times New Roman", Times, serif;
}
                          
.p2 {
    font-family: Arial, Helvetica, sans-serif;
}
                          
.p3 {
    font-family: "Lucida Console", "Courier New", monospace;
}

Стилізація шрифтів

Font Style

The font-style property is mostly used to specify italic text.

This property has three values:

  • normal - The text is shown normally
  • italic - The text is shown in italics
  • oblique - The text is "leaning" (oblique is very similar to italic, but less supported)

p.normal {
    font-style: normal;
}
              
p.italic {
    font-style: italic;
}
              
p.oblique {
    font-style: oblique;
}

Font Weight

Властивість font-weight використовується для налаштування жирності шрифту

.bold {
    font-weight: bold;
}

.weight100 {
    font-weight: 100;
}

Цей текст має normal (звичайну) вагу шрифту.

Цей текст має bold (жирний) шрифт.

Цей текст має bolder (більш жирний) шрифт.

Цей текст має lighter (більш легкий) шрифт.

Цей текст має font-weight 100 (найбільш легкий).

Цей текст має font-weight 900 (найбільш жирний).

Розмір шрифту

Властивість font-size визначає розмір тексту.

Здатність керувати розміром тексту важлива в веб-дизайні. Однак не слід використовувати зміни розміру шрифту, щоб робити абзаци схожими на заголовки, або заголовки схожими на абзаци.

Завжди використовуйте відповідні HTML теги, такі як <h1> - <h6> для заголовків та <p> для абзаців.

Абсолютний розмір:

  • Встановлює текст на заданий розмір
  • Не дозволяє користувачеві змінювати розмір тексту в усіх браузерах (погано для доступності)
  • Корисний, коли фізичний розмір вихідного пристрою відомий

Відносний розмір:

  • Встановлює розмір, відносно оточуючих елементів
  • Дозволяє користувачеві змінювати розмір тексту в браузерах
h1 {
    font-size: 40px;
}
              
h2 {
    font-size: 30px;
}
              
p {
    font-size: 14px;
}

Встановлення розміру шрифту за допомогою Em

Щоб дозволити користувачам змінювати розмір тексту (через меню браузера), багато розробників використовують em замість пікселів.

1em дорівнює поточному розміру шрифту. За замовчуванням розмір тексту в браузерах становить 16px. Тому стандартний розмір 1em — 16px.

Розмір можна обчислити з пікселів в em за допомогою цієї формули: пікселі / 16 = em

h1 {
    font-size: 2.5em; /* 40px/16=2.5em */
}
      
h2 {
    font-size: 1.875em; /* 30px/16=1.875em */
}
      
p {
    font-size: 0.875em; /* 14px/16=0.875em */
}

У наведеному вище прикладі розмір тексту в em такий самий, як у попередньому прикладі в пікселях. Однак із використанням розміру em можна налаштувати розмір тексту в усіх браузерах.

Одиниці вимірювань у CSS

Важливо: будь-які одиниці вимірювань пишуться разом із цифрами. Наприклад, вірно: 16px.

Абсолютні одиниці:

  • cm — сантиметри
  • mm — міліметри
  • in — дюйми
  • pt — пункти (1/72 дюйма)
  • pc — піки (1/6 дюйма)
  • px — пікселі (найпоширеніша одиниця)

Відносні одиниці:

  • em — розмір шрифту батьківського елемента
  • rem — розмір шрифту кореневого елемента (<html>)
  • % — відсотки, розмір залежить від батьківського елемента
  • ex — відносно висоти символу x шрифту, використовується рідко
  • vw — 1% від ширини області перегляду
  • vh — 1% від висоти області перегляду
  • vmin — мінімум між vw і vh
  • vmax — максимум між vw і vh

Приклад використання em і %:

1em та 100% — це розмір шрифту батьківського елемента. Наприклад:

body { 
    font-size: 14px; 
}
h1 { 
    font-size: 3em; 
}  
/* Розмір шрифту буде 42px */

Для h1 з font-size: 300% буде розраховано так: 14px * 300% / 100% = 42px.
